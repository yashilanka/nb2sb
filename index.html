Nb2Sb

Nb2Sb (Navbar to Sidebar) is a Bootstrap-only jQuery plug-in that converts your navbar into a sidebar.

Resize your window or open this page on a mobile browser.

This is a beta version. Many features aren't available yet
Dropdown menus aren't fully supported yet.

ABOUT

It's all about IDs and Classes!

The full set up guide is uploaded on Github

I added just a random id to the bootstrap button. (#nb2sb-btn)

<!-- icon/button -->
<button id="nb2sb-btn" type="button" class="navbar-toggle" data-toggle="collapse">
  <span class="icon-bar"></span>
  <span class="icon-bar"></span>
  <span class="icon-bar"></span> 
</button>

Then I added another random ID to the navbar content just before the ul: #nb2sb-content.

And eventually I added a random class to all elements that, when clicked, will close the sidebar (.close-sb) - This doesn't happen with the default navbar.

<!-- content -->
<div id="nb2sb-content">
  <ul class="nav navbar-nav nav-pills navbar-right">
    <li class="active"><a data-toggle="pill" href="#">#<span></a></li>
    <li><a data-toggle="pill" class="close-sb" href="#">#</a></li>
    <li><a data-toggle="pill" class="close-sb" href="#">#</a></li>
    <li><a data-toggle="pill" class="close-sb" href="#">#</a></li>
  </ul>
</div>

Finally I used those IDs and Class in the function.

To read the full list of options, please, visit the Github page

$( document ).ready(function() {
  $( '#nb2sb-nav' ).nb2sb({
    selectors: {
      opener: '#nb2sb-btn',
      content: '#nb2sb-content',
      closingLinks: '.close-sb'
    },
    settings: {
      animation: {
        easing: 'easeOutQuint'
      }				
    }
  });
});
